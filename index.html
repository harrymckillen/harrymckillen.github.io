This is will just contain a link to a bunch of projects I've hosted on github. Currently I have a small number of little projects I'm working on. They are as follows;

DOS-UI

  • A small CSS & AngularJS based framework. It can be broken apart purely for the styles, but I wish to add some handy Angular directives which can just be quickly used to generate DOS UI components for use in your projects.

YAFGS - Yet Another Feckin' Grid System

  • Combines a bunch of types of grid systems in one. Sometimes I find there is a lack of options if you choose one over the other. This aims to give loads and loads of options.

Irish Glyph Font

  • Just a simple font I created. 33 glyphs altogether, one for each county and the map of Ireland. Useful if you want to great a nice crisp map that will look great on all devices, especially with retina/hi-res screens.
